# Билеты по функциональному анализу

В каталоге собраны ответы на все 73 экзаменационных билета по функциональному
анализу. Материал разбит на семь логических частей и оформлен в виде HTML-страниц
с формулами, определениями, примерами и необходимыми доказательствами.

Основная точка входа — [`index.html`](index.html). Из оглавления можно перейти
к любой тематической части:

1. `part1-metric.html` — метрические пространства, полнота, теорема Бэра,
   пополнение и компактность.
2. `part2-linear.html` — линейные пространства, выпуклость, функционал
   Минковского и теорема Хана–Банаха.
3. `part3-normed.html` — нормированные, банаховы и евклидовы пространства.
4. `part4-hilbert.html` — ортогональные системы, ряды Фурье и гильбертовы
   пространства.
5. `part5-functionals.html` — топологические линейные пространства,
   непрерывные функционалы и сопряжённые пространства.
6. `part6-operators.html` — линейные операторы, операторная норма,
   обратимость и ряд Неймана.
7. `part7-spectrum-compact.html` — матричные производные, сопряжённые
   операторы, спектр и компактные операторы.

Материалы подготовлены по официальному
[`списку тем`](список_тем_v1.pdf), книге
«Элементы теории функций и функционального анализа» А. Н. Колмогорова и
С. В. Фомина, а также листкам домашних заданий.

Если в списке тем явно требуется доказательство, оно приведено полностью и,
в последних частях, разбито на последовательные шаги с пояснением их цели.
Для остальных теорем основное внимание уделено точной формулировке, смыслу
и применению.

Формулы отображаются через MathJax. Для корректного просмотра HTML-страниц
требуется подключение к интернету.
