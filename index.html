Как пользоваться. Раздел разбит по тематическим блокам. Формулы рендерятся через MathJax (нужен интернет для подгрузки библиотеки). Теоремы с пометкой «(с доказательством)» приведены полностью, остальные — формулировка и ключевая идея.

Самопроверка. Отдельный набор обязательных вопросов по всем билетам повторяет полные формулировки тем из официального списка.

Согласование с конспектом.
